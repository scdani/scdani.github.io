HOW

Architecture

Every Claude Code request goes through a two-layer proxy before reaching the model.

Request flow

  1. Claude Code sends every request to localhost:4140 — the Python/Flask proxy
  2. The proxy inspects the content: contains an image? → route to GEP  •  text or code? → route to copilot-api on :4141
  3. Images GEP (EU first, US fallback) handles vision and responds directly — does not go via GitHub Copilot
    Text / code copilot-api translates to GitHub Copilot API format and forwards to GitHub cloud
  4. GitHub Copilot cloud processes text/code requests and returns a response using Claude Opus, Sonnet, or Haiku (or any other configured model)
  5. Both paths converge at the Response node and route back to Claude Code — no separate Anthropic account or billing needed

Models available via your Copilot license

Claude Code is pre-configured to use the three Claude models below. Your Copilot license gives you access to many more — if you want to use a different model, you can add it to your settings.json and Claude Code will route to it via GitHub Copilot.

Installed files

What the setup creates

The automated guide installs these 8 files into your ~/.claude/ directory.

📄
scripts/vision-proxy.py
Flask server on :4140. Routes image requests to GEP EU/US, text to copilot-api. Contains your GEP keys.
📄
scripts/vision-proxy-wrapper.ps1
Starts vision-proxy.py, handles PID tracking and restarts.
📄
scripts/copilot-proxy-wrapper.ps1
Starts copilot-api on :4141, sets required env vars, logs output.
📄
scripts/ensure-proxies.ps1
Checks if both proxies are running and starts any that have stopped. Run by Task Scheduler at login.
📄
scripts/start-copilot-proxy.sh
Bash entrypoint for starting the full proxy stack manually from Git Bash.
📄
scripts/check-copilot-proxy.sh
Health check script. Verifies both proxies respond, tests model routing.
⚙️
settings.json
Claude Code env config: ANTHROPIC_BASE_URL=http://localhost:4140, model version mappings, API key placeholder.
📝
CLAUDE.md
Global instructions injected into every Claude Code session. Includes model awareness and proxy routing notes.
